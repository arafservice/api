# api
api خوش امدگویی

یک API ساده که با استفاده از Flask نوشته شده و قابلیت‌های زیر را ارائه می‌دهد:
*   نمایش پیام خوشامدگویی.
*   دریافت نام و ارسال پیام خوشامدگویی شخصی‌سازی شده.
*   جمع کردن دو عدد که از طریق درخواست POST ارسال می‌شوند.

## معرفی

این پروژه یک نمونه API سبک وزن با پایتون است که برای نمایش نحوه ساخت و استقرار APIهای ساده طراحی شده است. این API با فریم‌ورک Flask ساخته شده و دو نقطه پایانی (endpoint) اصلی دارد.

## ویژگی‌ها

*   **پیام خوشامدگویی عمومی:** نقطه پایانی `/` یک پیام خوشامدگویی اولیه را نمایش می‌دهد.
*   **پیام خوشامدگویی شخصی:** نقطه پایانی `/greet/<string:name>` (با متد GET) نام شما را دریافت کرده و یک پیام خوشامدگویی شخصی‌سازی شده برمی‌گرداند.
*   **عملیات جمع:** نقطه پایانی `/add` (با متد POST) دو عدد را از بدنه درخواست JSON دریافت کرده و حاصل جمع آن‌ها را برمی‌گرداند.
